# Tênis Linhares - versão final v9

Versão consolidada com:
- logo definitiva
- verde fluorescente e branco
- check-in com mensagens amigáveis
- inscrição em torneios com categorias prontas
- painel admin organizado
- confirmações agrupadas por data e horário
- inscrições agrupadas por evento e categoria
- eventos sem exemplo automático
- botões verdes para copiar PIX
- sem exibição de HTML bruto na tela

## Arquivos
- app.py
- schema.sql
- requirements.txt
- assets/logo.jpeg

## Publicação
1. Suba `app.py`, `schema.sql`, `requirements.txt`, `README.md` e `assets`
2. Rode `schema.sql` no Supabase
3. Reboot no app do Streamlit
