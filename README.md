# Tênis Linhares - versão final

Versão baseada na v11 aprovada, mantendo o design e as funcionalidades principais.

Inclui:
- Check-in de aulas com mensagem de confirmação.
- Solicitação de reposição de aula.
- Eventos/torneios com inscrição por categoria.
- Financeiro com PIX e botão de copiar.
- Painel administrativo completo no menu lateral.
- Administração de alunos, eventos, inscrições, confirmações e reposições.
- Filtros nas listas administrativas.
- Logo `logo.jpeg` na raiz e também suporte a `assets/logo.jpeg`.

## Publicação
1. Suba `app.py`, `requirements.txt`, `schema.sql`, `README.md` e `logo.jpeg` no GitHub.
2. Rode `schema.sql` no Supabase.
3. No Streamlit, faça Reboot app e Clear cache.
